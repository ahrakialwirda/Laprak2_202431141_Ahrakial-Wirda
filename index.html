a. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web modern?

CSS digunakan untuk mengatur tampilan agar lebih menarik, terstruktur, dan profesional. JavaScript digunakan untuk membuat halaman menjadi interaktif dan dinamis sehingga pengalaman pengguna menjadi lebih baik.

b. Jelaskan konsep dan tujuan utama dari Responsive Web Design (RWD)!

RWD adalah teknik desain web agar tampilan menyesuaikan ukuran layar (PC, tablet, smartphone). Tujuannya agar website tetap nyaman digunakan dalam berbagai perangkat tanpa perlu membuat versi terpisah.

c. Bagaimana CSS, JavaScript, dan RWD saling melengkapi?

CSS mengatur desain, JavaScript menambah interaktivitas, dan RWD memastikan tampilan tetap nyaman di berbagai perangkat. Ketiganya bekerja bersama untuk menciptakan website yang adaptif, modern, dan user-friendly.
